## [Note] What is Vim?

`Vim` 是 `Vi + Improved` 的縮寫，是十分受歡迎的編輯器之一，學習使用 Vim 有許多好處，包括但不限於：

- Vim 的方向操作可以在 `man` 或 `less` 命令下使用
- 跨作業系統使用，幾乎所有的系統都能夠使用 Vim
- 提供圖形化介面（GUI）與命列行介面（TUI）
