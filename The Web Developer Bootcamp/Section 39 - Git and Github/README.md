## 362, Intro To Git

## 363, Git Init, Add, and Commit

- `git init` 在當前目錄下初始化 git 來追蹤版本。
- `git add` 添加檔案或資料夾到追蹤的清單中。
- `git commit` 當完成一個版本之後提交對象。

## 364, Git Log and Checkout

- `git log` 查看提交的版本紀錄。
- `git checkout` 切換或創建分支（branch）。
