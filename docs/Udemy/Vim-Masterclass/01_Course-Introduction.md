# Course Introduction

## What is Vim?

![Vim Logo](https://user-images.githubusercontent.com/26391143/78927477-8ec12e80-7ad1-11ea-9de0-0bbde38caf05.png =200x)

Vim 源自於其正式名稱 `Vi IMproved` 的縮寫，是從純文字編輯器 [vi](http://ex-vi.sourceforge.net/) 而來，目前和 [Emacs](https://www.gnu.org/software/emacs/) 並列為最受 Unix-Like 系統使用者所喜愛的編輯器。

學習使用 Vim 有許多好處，比如說：

- 在 Vim 中使用的方向操作可以在 `man` 或 `less` 命令下使用
- 可以跨作業系統使用，幾乎所有的系統都能夠使用 Vim
- 提供圖形化介面與命列行介面
- 相關的命令和快捷鍵容易記憶

## Install Vim

### Arch Linux

```bash
$ sudo pacman -S vim
$ sudo pacman -S gvim
```

### mac OS

```bash
$ brew install vim
```
