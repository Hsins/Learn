# Section 10 - Arrow Functions

## 027, Fat Arrow Functions

在 ES6 中 **箭頭函數（Arrow Function）** 也是一項重大變革，在這一單元中將會介紹什麼時候該使用他。首先，先來看一下函數定義方式的差別：

```javascript
// Original Function
const add = function(a, b) {
  return a + b;
}

// Arrow Function
const add = (a, b) => {
  return a + b;
}
```
